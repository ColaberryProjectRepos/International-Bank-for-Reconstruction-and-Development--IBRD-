Create the 3 Database that is required for this project

The 3 Database are:

  • International Bank for Reconstruction and Development (IBRD) DW
  • International Bank for Reconstruction and Development (IBRD) ODS
  • International Bank for Reconstruction and Development (IBRD) Staging

The following steps will be performed in SQL Server Management Studio (SSMS) to create the database

  1. Right Click on the Database, then click on New Database.
  2. Name the Database.
  3. Click Ok to create the Database, then refresh Database to locate New Database Created.
